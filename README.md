### 上传报关单通讯回执

**url：**`/formResult/upload/tongXun?ediNo=?`

**type：**`POST`

**请求样例：**

```json
{
    "channel": "",
    "note": ""
}
```

**参数说明：**

|节点名|是否必填|说明|
|----|----|----|
|ediNo|是|ediNo编号|
|channel|是|回执类型|
|note|是|备注|
